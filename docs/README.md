# HawZZ 的个人主页

欢迎来到 **PersonalLib**。这里既是我的个人主页，也是我持续维护的知识库。

我会把技术学习、项目经验、工作思考和长期可复用的资料沉淀在这里，让它们可以被检索、被链接，也可以随着实践不断更新。

## 关注方向

- 车载边缘计算、车联网与智能交通相关技术
- 工业通信协议与工程实践，例如 Modbus
- 产品、架构、系统设计和工作中的复盘记录
- 值得长期保存的学习资料、概念解释和问题清单
